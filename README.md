# AI Kubernetes Troubleshooting Agent

This is an on-demand Kubernetes troubleshooting system that identifies root causes and suggests fixes using an AI Kubernetes Agent.

## Project Structure

```text
ai-kubernetes-agent/
├── backend/            # FastAPI Backend (Orchestrator)
├── frontend/           # Next.js Frontend (TS, Tailwind)
├── docs/               # Documentation
├── prompts/            # Prompt templates
├── docker-compose.yml  # Docker Compose configuration
└── README.md
```

## Running the Project

Ensure you have Docker and Docker Compose installed, then run:

```bash
docker compose up --build
```

Access the services at:
- Frontend: [http://localhost:3000](http://localhost:3000)
- Backend Health: [http://localhost:8000/health](http://localhost:8000/health)
